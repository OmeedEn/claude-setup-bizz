# Luxe & Co — Demo Agent

This is a **demo agent** for sales calls. It showcases what a Professional-tier Claude bot looks like for an e-commerce business.

## How to Use This Demo

1. Open Claude Code or Claude Desktop
2. Point it at this directory (or copy the CLAUDE.md into a project)
3. Start asking questions as if you're a customer

### Demo Scenarios to Walk Through

**Order Tracking:**
> "Hi, I placed an order last week. Order #1042. Can you check the status?"

**Product Recommendation:**
> "I'm looking for a housewarming gift for my sister. Budget around $100-$150."

**Return Request:**
> "I received the Cloud Duvet Set but I ordered Queen and got King. I need to exchange it."

**Product Question:**
> "Are your candles made with real essential oils? How long do they burn?"

**Promo Code:**
> "Do you have any discounts right now? It's my first time shopping here."

## What to Highlight During Sales Calls

1. **It knows the entire product catalog** — no training a new hire
2. **Consistent brand voice** — every customer gets the same quality experience
3. **Handles the common stuff automatically** — order tracking, returns, FAQs
4. **Knows when to escalate** — doesn't try to handle things it shouldn't
5. **24/7 availability** — never misses a customer message
6. **Connects to real tools** — Shopify, Stripe, inventory (show the mcp-config.json)
