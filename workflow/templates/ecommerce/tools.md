# E-Commerce Integration Notes

## Shopify
- Store admin access needed (API key + secret)
- Scopes required: read_orders, write_orders, read_products, read_customers, read_inventory
- Webhook setup for real-time order updates (optional)

## Stripe
- Restricted API key recommended (not full secret key)
- Permissions: read charges, create refunds, read customers
- Test mode keys for development

## Common Customizations
- Custom product recommendation logic
- Integration with email marketing (Klaviyo, Mailchimp)
- Live chat widget connection
- Social media DM integration

## Knowledge Base Documents to Collect
- Product catalog / descriptions
- Shipping policy
- Return/exchange policy
- FAQ document
- Sizing guides (if apparel)
- Promotional/sale information
